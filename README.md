Platform-ready profile photos, cover images, and logos for every social media platform. One folder per platform, grab what you need and upload.

## How it was made

- Headshots: top-anchored crop from AI-generated portrait, upscaled to 4096x4096, LANCZOS downscaled per platform
- Cover photos: cairocffi rendered (text as vector paths), 8K supersampled, all 5 visual effects (text shadow, gradient, underline, headshot glow, ambient glow)
- EXIF metadata embedded in all JPEG headshots (Title, Description, Copyright, Artist)
- All images optimized for each platform's upload requirements
